\### Флоу работы с Git



1\. \\\*Cоздать папку под проект\\\*

2\. \\\*Перейти в папку\\\*

3\. \\\*git init - сделать папку репозиторием\\\*

4\. \\\* rm -rf .git - удалить папку .git если ошибся \\\*

5\. \\\* git status - показать текущее состояние репозитория\\\*

6\. \\\* git add --all - подготовить файлы к сохранению \\\*

7\. \\\* git commit -m "comment" - выполнить коммит \\\*

8\. \\\* git log - посмотреть историю коммитов\\\*

9\. \\\* Создать удаленный репозиторий на GitHub\\\*

10\. \\\* git remote add origin git@github.com:Danilamaikl/second-project.git\\\*

11\. \\\* git remote -v - убедиться, что репозиторий привязан\\\*

12\. \\\* git push -u origin main\\\*
13. \\\* Проверить все ли корректно на GitHub\\\*
